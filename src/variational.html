River Level Predictions

The forecasts/predictions are via Structural Time Series + Variational Inference. Each graph's title denotes Predictions: River Level Gauge Location/Catchment Area. Each river level measuring device is located within, assigned to, a catchment area. An upcoming map will help, additionally SEPA (Scottish Environment Protection Agency) provides gauge station location details/maps.

The river level unit of measure is metres ($m$). The river level values are the values relative to Ordnance Datum, i.e., relative water level + gauge datum.

Prediction errors with respect to model-training-data are expected to be small. The testing phase observations, i.e., Ground Truth (TE), were not used for training. Hence, the predictions relative to the testing phase observations hint at model performance. There are more metrics within the Project/Model Details pages.
